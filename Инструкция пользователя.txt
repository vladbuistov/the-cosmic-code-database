ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ:

1. Чтобы начать работу с базой данных, нужно нажать на кнопку "Выберите файл". 
Данная кнопка располагается в левом верхнем углу главного окна программы.

2. После этого Вам нужно выбрать базу данных, с которой Вы хотите работать.

3. Следующим действием будет выбор таблицы, которую Вы будете изменять. 
Для этого нужно развернуть список таблиц, находящихся в выбранной базе данных, нажатием на стрелку выпадающего списка. 
Далее выбираем нужную таблицу.

4. Теперь мы можем добавлять, изменять и удалять записи в выбранной таблице:

- Для добавления новой записи в таблицу выберите кнопку "Добавить запись". Она расположена в левом нижнем углу главного окна программы.
После нажатия на данную кнопку откроется новое окно. В этом окне необходимо ввести данные в соответствующие поля, расположенные справа от названий столбцов. 
После ввода данных требуется нажать кнопку "Добавить". Новая запись появится в таблице.

- Для изменения существующей записи выберите кнопку "Обновить запись". Она расположена в нижней части по центру главного окна программы.
После нажатия на данную кнопку откроется новое окно. В нём необходимо указать id нужной записи и ввести изменения в соответствующие поля. 
Чтобы сохранить изменения, нажимаем кнопку "Обновить". Запись изменена.

- Для удаления записи выберите кнопку "Удалить запись". Она расположена в правом нижнем углу главного окна программы.
После нажатия на данную кнопку откроется новое окно. Чтобы удалить запись, нужно указать её идентификатор и нажать на кнопку "Удалить". Запись будет удалена.
